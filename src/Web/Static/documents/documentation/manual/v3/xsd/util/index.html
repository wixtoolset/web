The elements defined in the Utility Extension.

In order to use this schema, you should:

  1. Add Reference in the WiX project to <WiX_install_folder>\bin\WixUtilExtension.dll. If you are using light.exe directly, use the -ext parameter.
  2. Add xmlns:util="http://schemas.microsoft.com/wix/UtilExtension" attribute to the root element of the XML file.
  3. Add the util: prefix to each element name as you use it, e.g. util:CloseApplication.
