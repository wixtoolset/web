title: VersionType (Simple Type)

---
<dl>
  <dt>Description</dt>
  <dd>Values of this type will look like: "x.x.x.x" where x is an integer from 0 to 65534.</dd>
  <dt>Pattern Type</dt>
  <dd>Must match the regular expression: '(\d{1,5}\.){3}\d{1,5}'.</dd>
